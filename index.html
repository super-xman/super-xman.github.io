项目一：音乐指挥训练与演奏

  这是我独立开发的第一个落地项目，面向音乐指挥专业学生研发，帮助学生进行音乐指挥训练和现场演奏模拟。最终效果得到了音乐学院多位老师的认可，顺利通过最终验收。这款产品对指挥教育行业具有重大意义，据音乐学院院长透露，培养一名指挥系的学生需要同时三名老师配合其指挥动作进行演奏，教育成本非常高，而这款产品可以使学生脱离老师的配合，用软件生成对应节奏的音乐来响应学生的指挥动作，且目前已经达到了精准契合的程度，开创了指挥控制合成音乐的先河。因为没有导师和公司的授权，这里只能展示录屏资源，请谅解。

视频教学

视频教学分为三个专项模块，其中指挥图示模块支持显示轨迹。

辅助训练

根据指挥轨迹，实时识别指挥意图，并评估动作与音乐的契合度。指挥结束后用户可以选择保存指挥轨迹或是重新开始。轨迹数据会以JSON格式保存在本地，并支持回放。回放依然会评估轨迹，方便指挥者事后总结和纠正。

模拟演奏

指挥能够通过指挥动作控制音乐的节奏、强弱等，并且输出的的音乐与动作节拍高度一致。指挥的幅度越大，音乐音量越高；指挥的速度越大，音乐节奏越快（midi音乐节奏加快不会发生变调）。

项目二：场景化单词记忆

  这是由我独立开发的第一款 web app，也可以算是我的启蒙老师，帮助我在实践中巩固了前端三件套的基本语法，带我进入了前端的世界。当初的设想是做一款涵盖中小学常用单词的 web app，特点是场景化记忆和交互记忆，支持生词本记录。但最终由于时间问题，只做出了设想中的一部分功能。当时也没考虑到适配不同窗口大小，这是一个遗憾，有空会好好完善一下，目前仅建议在桌面端浏览器操作此页面。

书房场景

拖动单词标签到正确位置即可得分，得分递减会随错误次数递减。

器官场景

拖拽到正确位置时会去除相应部分的模糊特效，身体场景全部回答正确后会进入五官场景。

方块操作

按钮调整方块的尺寸位置颜色等，学习几何特征相关的形容词。

自定义场景

自己选择图片作为场景，并支持自己定义单词标签和对应位置。

课程表

拖拽元素到正确位置则会“粘贴”到课程表上，课程单词能重复使用，创建自己的课程表。

全家福

拖拽元素到家谱树上进行配对，帮助学习与亲人相关的单词。

实战演练

在这里可以亲身体验上述视频里的小游戏。由于没有适配不同屏幕尺寸，可能会遇到窗口显示不完整的情况，建议使用桌面浏览器。目前只实现了 primary shool 和 lab 模块，见谅。
